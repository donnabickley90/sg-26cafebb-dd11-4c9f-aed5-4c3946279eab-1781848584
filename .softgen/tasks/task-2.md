## Notes
The Dashboard is the command centre of the planner. It must show a highly detailed snapshot of everything due today at a glance, incorporating progress rings and quick-action buttons.

## Checklist
- [x] Display today's date and day of the week prominently at the top.
- [x] Add a "Quick Actions" button row: Open today's planner, Start daily reset, Start weekly reset, Add task, Add meal, Add birthday, Add quick note, Start declutter challenge, Open deep clean calendar.
- [x] Create a "Today's Schedule" widget showing upcoming hourly blocks.
- [x] Add widgets for: Today's Meals, Chores due today, Deep clean tasks due today, Upcoming birthdays, Important dates/anniversaries.
- [x] Include interactive widgets for: Top 3 priorities, Quick notes, Mood tracker, and Energy/Spoons tracker.
- [x] Build a "Progress Summary" section using rings/bars showing: Daily planner completion percentage, Cleaning progress percentage, and Current streaks.

## Acceptance
- Opening the app lands on the Dashboard showing all today's data.
- Quick action buttons successfully open modals or navigate to their respective flows.
- Progress rings visually fill up as simulated tasks are checked off.
