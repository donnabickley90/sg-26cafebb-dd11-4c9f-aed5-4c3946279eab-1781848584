## Notes
A dedicated calendar specifically for heavy, infrequent cleaning tasks, preventing them from cluttering the daily/weekly routine.

## Checklist
- [ ] Build a standalone Deep Clean Calendar view showing scheduled deep clean jobs.
- [ ] Create an "Unscheduled Deep Clean Tasks" backlog list alongside the calendar.
- [ ] Pre-populate the unscheduled list with the requested defaults (e.g., Clean oven, Wash curtains, Rotate mattress, Scrub grout, Clean exhaust fan).
- [ ] Allow users to drag-and-drop or select a date to assign an unscheduled task to a specific day.
- [ ] Add ability to reschedule or mark tasks complete directly from the calendar view.
- [ ] Implement filtering to view upcoming deep clean jobs by specific Room.
- [ ] Display an alert/badge for Overdue deep clean tasks.

## Acceptance
- User can view their unscheduled backlog, click "Scrub grout", and assign it to a Saturday.
- The scheduled deep clean task appears on the main Dashboard on the day it is due.
